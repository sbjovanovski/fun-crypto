## Run locally
```
$ yarn
$ yarn dev
```
Open `localhost:3000`

## Libraries used
- **tailwindcss** for CSS
- **dotenv** for Environment variables
- **typescript** for Typings
- **biome** for Linting and code formatting
- **esbuild** for Building the project and running it locally
